eslint 可共享配置学习
## 安装
> npm i @smfdnx/test-eslint-vue


## package属性：
> description：包的简介
> repository：适合写上 GitHub 地址 — 所以你可以写成这种格式 username/repo
> license：这里是 MIT 认证
> main：包的入口文件，位置在文件夹的根目录
> keywords：添加一些关键词更容易使你的包被搜索到

## 更新：
> npm version x.x.x
> npm version patch更新一个补丁
> npm version minor更新一个小改动
> npm version major更新一个大改动
